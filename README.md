# Setting Up

Compilation

Use the Makefile to compile the project:

- make → compile  
- make fclean → clean all unnecessary files (including the executable)  
- make re → clean and re-compile  

Usage

Once the code is compiled, the executable can be used in several ways:

- Using a map file:
./setting_up "path/to/map_file"

- Using a number of lines and a pattern:
./setting_up 12 "......o.."
